# CC Roast

**Your CLAUDE.md, brutally honest.**

Paste your Claude Code instruction file and get a comedy-style code review. 11 verdict types, 4 score dimensions, and roasts that actually teach you something.

**[Get Roasted](https://yurukusa.github.io/cc-roast/)**

## How It Works

1. Paste your `CLAUDE.md` contents
2. Click "Roast My Config"
3. Get your verdict, scores, and personalized roasts
4. Share the results (if you dare)

## Verdict Types

| Verdict | What it means |
|---------|--------------|
| S-Tier Micromanager | 25+ strict rules in 200+ lines |
| Helicopter Operator | 15+ strict directives |
| The Novelist | 500+ lines of instructions |
| Security Paranoid | Fort Knox-level lockdown |
| Overengineered Perfection | More architecture than most apps |
| Vibes Only | Barely any rules at all |
| The Minimalist | Clean and lean |
| Sticky Note Energy | Under 15 lines |
| Well-Balanced Config | The Goldilocks zone |
| Trust Fund AI | $200/month + a post-it note |
| Standard Issue Operator | Just... fine |

## Score Dimensions

- **Verbosity** — How long is your novel?
- **Strictness** — How many NEVER/DO NOT/ALWAYS/MUST?
- **Complexity** — Hooks, MCP, code blocks, tables
- **Thoroughness** — Git safety, testing, security coverage
